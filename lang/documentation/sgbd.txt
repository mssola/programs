Configuració de SGBD
====================


PostgreSQL
==========

Per crear el primer usuari:

  * El daemon ha d'estar en execució.
  * su root
  * su - postgres
  * createuser [-DRSP] <username>
    -> D: no pot crear bases de dades.
    -> R: no pot crear rols.
    -> S: no pot ser superusuari.
    -> P: demana la password.

Crear base de dades (amb tots els privilegis):

  * createdb -O fiqs fiqsdb
  * grant all privilegis on nom-taula to nom-usuari

Macros especials:

  * \l: llista bases de dades.
  * \c nom-bd: connectar a una bd en particular.
  * \d: llista taules de la base de dades.
  * \d nom-taula: descripció de la taula.

Activar suport per PHP:

  * vim /etc/php/php.ini
  * Descomentar (;) línies: extension=pgsql.so i
    extension=pdo.so i extension=pdo_pgsql.so si es necessita pdo.
  * Fer un restart de l'Apache.


MySQL
=====

Per crear el primer usuari:

  * El daemon ha d'estar en execució.
  * mysql -p -u root (sense ficar password).
  * create user 'nom-usuari'@'localhost' identified by 'contrasenya';

Crear base de dades (amb tots els privilegis):

  * create database nom-bd;
  * grant all privileges on nom-bd.* to "nom-usuaru"@"localhost" identified by "";

Consultes especials:

  * show tables;  -> mostra les taules.
  * select host, user, password from mysql.user;   -> consulta usuaris
